## 3) Talk Track / Script (10–15 lines)

### Opening
- “I want to talk about something difficult. I’m going to be direct, and I also care about you.”

### Deliver the key message (then pause)
- “<Key message>.”
- (Pause. Let them react.)

### Evidence + impact (2–5 bullets max)
- “On <date>, <observation>. The impact was <impact>.”

### Ask/decision + support
- “Going forward, I need <request> by <date>.”
- “To support you, I can <support>.”

### Invite understanding (not debate)
- “What are you hearing me say?”
- “What feels unclear right now?”

### Close
- “To summarize: <summary>.”
- “Next step is <next step> on <date>.”

## 4) Objections + Emotion Handling Plan

| Likely reaction | What they might say | What you do (1–2 moves) | What you say (1–3 sentences) | What NOT to do |
|---|---|---|---|---|
| Shock |  | Pause; acknowledge |  | Over-explain |
| Anger |  | Breathe; reflect |  | Argue |
| Sadness |  | Silence; offer time |  | Rush |
| Defensiveness |  | Return to facts |  | Pile on |
| Negotiation |  | Re-state non-negotiable |  | Make policy promises |

## Phrase bank (short)

Direct + caring
- “I’m going to be direct because I care about your success here.”
- “This is hard to say, and I want to be respectful and clear.”

Return to the core message
- “I hear you. I want to stay on the main point: …”
- “Let’s focus on what happens next.”

Feelings (examples)
- “I felt disappointed / concerned / surprised / frustrated when …”
