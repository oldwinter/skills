# Rubric (score the Trade-off Evaluation Pack)

Score each category 0/1/2:
- 0 = missing / unusable
- 1 = present but incomplete
- 2 = clear, decision-ready, and internally consistent

Suggested passing bar: **>= 10/12**

## 1) Framing + options (0–2)
- Decision is crisp, dated, and has an owner
- Options are concrete and mutually distinguishable

## 2) Criteria + horizon (0–2)
- 4–8 criteria (incl. guardrail) + explicit time horizon
- “Not optimizing for” is stated to prevent goalpost moving

## 3) All-in cost + opportunity cost (0–2)
- Costs are complete (incl. hidden costs) and comparable
- Opportunity cost (displaced work) is explicit

## 4) Impact ranges + assumptions (0–2)
- Ranges + confidence are used; no false precision
- Top assumptions driving the decision are clear

## 5) Thought experiments + validation plan (0–2)
- Pre-mortem and failure modes are captured
- Minimal validation plan exists when it could change the decision

## 6) Recommendation quality (0–2)
- Recommendation is explicit with rationale and trade-offs
- Includes **Risks / Open questions / Next steps** + a review date / triggers
