# Rubric (Score 1–5)

Score the Managing Up Operating System Pack. Target: **≥ 20/25**, with no category below **3**.

## 1) Goal clarity + boundaries (1–5)
1: Vague advice; unclear success definition; ignores HR/safety boundaries.  
3: Goals mostly clear; basic boundaries and constraints acknowledged.  
5: Crisp goals + 4-week success definition; clear “when not to use” and escalation boundaries.

## 2) Manager profile usefulness (1–5)
1: Generic personality notes; no implications.  
3: Some actionable insights; partially validated.  
5: Clear optimization model + comms preferences + decision style; 3+ explicit behavior changes.

## 3) Communication system quality (1–5)
1: No system; ad hoc updates.  
3: Weekly update and basic 1:1 plan exist.  
5: Clear “what goes where”, skimmable weekly update, and decision memo usage is well-defined.

## 4) Escalation + ask effectiveness (1–5)
1: Asks are vague; escalates too late or too often.  
3: Some triggers and asks; inconsistent specificity.  
5: Triggered escalation plan + ask ladder; asks are time-bounded and easy to action.

## 5) Boundaries + influence plan (1–5)
1: Avoids boundaries; no influence loop.  
3: Some scripts and a basic influence plan.  
5: Outcome-based boundary scripts + 30-day strategic contribution loop with pre-wiring and success signals.
