## 6) Rollout Plan (pilot → adoption)

**Phase 0: Prep (week 0)**
- [ ] Finalize Golden Rituals list (3–7 max)
- [ ] Create templates + doc locations
- [ ] Assign ritual owners

**Phase 1: Pilot (weeks 1–2)**
- [ ] Run 2–3 rituals with the smallest viable group
- [ ] Collect feedback (3 questions)
- [ ] Adjust templates and timing

**Phase 2: Full rollout (weeks 3–6)**
- [ ] Add calendar invites / recurring docs
- [ ] Publish onboarding primer
- [ ] Reinforce usage (“Bring X to <RitualName>”)

**Phase 3: Governance (ongoing)**
- [ ] Monthly pulse (1–3 questions)
- [ ] Quarterly ritual review (keep/change/kill)

---

## 7) Quarterly Ritual Review (agenda)

**Inputs:** ritual inventory + feedback + outcome signals

1) Reconfirm outcomes (5 min)
2) Review each ritual: keep/change/kill (2–5 min each)
3) Review time budget (5 min)
4) Decide changes + owners (10 min)
5) Update onboarding primer + ritual index (10 min)
