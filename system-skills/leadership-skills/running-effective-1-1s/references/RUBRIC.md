# Rubric (Score 1–5)

Score the 1:1 Operating System Pack. Target: **≥ 22/30**, with no category below **3**.

## 1) Scope + purpose clarity (1–5)
1: Unclear why 1:1s exist; no boundaries; turns into status.  
3: Purpose is mostly clear; boundaries exist; some channel separation.  
5: Crisp purpose + “what goes where” map; boundaries and escalation points are explicit.

## 2) Cadence + meeting design (1–5)
1: One-size-fits-all cadence; meeting bloat persists.  
3: Reasonable cadence; some flexibility; basic barbell idea.  
5: Cadence is tailored by relationship type and includes a clear urgent-meeting path.

## 3) Documentation + follow-through (1–5)
1: No shared system; action items disappear.  
3: Shared doc exists; some follow-through.  
5: Shared doc + topics backlog + action tracker; every meeting ends with owners and checkpoints.

## 4) Coaching effectiveness (1–5)
1: Manager advice-gives; report dependency grows.  
3: Some coaching prompts; occasional advice default.  
5: Coaching-first rules + question bank; clear exceptions; promotes independent problem solving.

## 5) Career development strength (1–5)
1: No career plan; tactical-only.  
3: Some growth discussion; limited follow-up.  
5: Dedicated life story/dreams/plan sequence with 1–3 growth bets and checkpoints.

## 6) Wellbeing + special situations (1–5)
1: Ignored or handled unsafely.  
3: Basic check-ins and some guidance.  
5: Respectful wellbeing pattern with boundaries/escalation + strong post-crisis and skip-level playbooks.
