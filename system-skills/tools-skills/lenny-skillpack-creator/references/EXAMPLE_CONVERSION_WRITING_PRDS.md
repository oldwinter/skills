# Example Conversion: “Writing PRDs” → Agent-Executable Pack (outline)

This is a partial example of how to refactor a Refound skill into an executable pack.

## 1) Identify pattern
- Doc Generator + AI Feature Pack

## 2) Output contract (deliverables)
- PR/FAQ
- PRD
- AI Eval Spec (if AI feature)
- Prompt Set (if AI feature)

## 3) Workflow (compressed)
1) Decide artifact set (PR/FAQ vs PRD vs AI pack)
2) Intake (ask max 5 questions)
3) Draft narrative (why/why now)
4) Define scope boundaries (goals/non-goals/out of scope)
5) Convert into numbered requirements (R1…)
6) Define success metrics + instrumentation
7) Risks + rollout + open questions
8) Quality gate + finalize
