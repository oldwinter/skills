# Rubric (1–5)

Score the Competitive Analysis Pack. A “ship-ready” pack is typically **≥ 22/30** with no category below 3.

## 1) Decision clarity (1–5)
- **1:** No decision stated; looks like general research notes.
- **3:** Decision and audience are stated; scope is mostly clear.
- **5:** Decision, deadline, and success criteria are explicit; pack is obviously decision-useful.

## 2) Alternatives coverage (1–5)
- **1:** Only direct competitors listed.
- **3:** Includes status quo and at least one non-obvious alternative.
- **5:** Clear competitive alternatives map that identifies the true deal alternative(s).

## 3) Evidence quality (1–5)
- **1:** Unsupported claims; no sources; certainty without proof.
- **3:** Some evidence; confidence labeling is inconsistent.
- **5:** Evidence-backed claims with confidence/unknowns explicit and no reliance on confidential info.

## 4) Customer POV + criteria (1–5)
- **1:** Feature checklist; little about customer outcomes.
- **3:** Criteria reflect customer needs but miss key decision drivers.
- **5:** Criteria + matrix clearly explain why each alternative wins/loses for the customer.

## 5) Differentiation + positioning (1–5)
- **1:** Generic value prop that could apply to anyone.
- **3:** Some differentiation, but not anchored to a specific alternative.
- **5:** 2–3 crisp positioning hypotheses explicitly “against X” with proof points and tradeoffs.

## 6) Actionability (1–5)
- **1:** No clear next actions; only observations.
- **3:** Some recommendations, but weak linkage to findings.
- **5:** Specific recommendations + battlecards + monitoring plan that can be executed immediately.
