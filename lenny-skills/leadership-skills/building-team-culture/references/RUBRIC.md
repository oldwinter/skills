# Rubric (Score 1–5)

Score the Team Culture Operating System Pack. Target: **≥ 24/35**, with no category below **3**.

## 1) Clarity of scope + target state (1–5)
1: Vague culture aspirations; no clear “more true in 8 weeks” target.  
3: Target behaviors are mostly clear; scope boundaries exist.  
5: Crisp target state with observable behaviors and explicit boundaries.

## 2) Evidence-based diagnosis (1–5)
1: Generic diagnosis; no examples.  
3: Some examples; plausible root causes.  
5: Strong examples; separates stated vs lived culture; sacred cows are identified.

## 3) Culture code quality (1–5)
1: Generic values with no behaviors.  
3: Principles include some behaviors and basic do/don’t.  
5: Principles are behavior-based with decision rules, anti-patterns, and measurable signals.

## 4) Operationalization (norms + rituals) (1–5)
1: No norms/rituals or they’re ceremonial.  
3: Some norms and rituals; ownership partially defined.  
5: Norms reduce ambiguity; rituals have owners, agendas, outputs, and map to principles.

## 5) Reinforcement plan realism (1–5)
1: “Announce the doc” plan; no ownership.  
3: Basic rollout steps; some reinforcement channels.  
5: Concrete 30/60/90 plan with coaching, onboarding/hiring hooks, and assigned DRIs.

## 6) Psychological safety + conflict handling (1–5)
1: Safety is mentioned but not operationalized.  
3: Includes some safety practices; conflict norms exist.  
5: Safety practices are concrete; leaders’ behaviors are specified; repair loops exist.

## 7) Measurement + iteration (1–5)
1: No measurement; no iteration plan.  
3: Some metrics or pulse questions; basic cadence.  
5: Leading indicators + pulse questions + iteration cadence tied to rituals and focus areas.
