# Rubric (Score 1–5)

Score the Promotion Case Pack. Target: **≥ 24/35**, with no category below **3**.

## 1) Target clarity + timing (1–5)
1: No explicit target level or promo window; unclear ask.  
3: Target and timing are stated; some process details known.  
5: Crisp target, why-now, and calendar/process constraints are explicit.

## 2) Ladder/expectations alignment (1–5)
1: No ladder mapping; claims are generic.  
3: Some mapping to competencies; behaviors partially specified.  
5: Clear competency map with observable next-level signals and explicit gaps/plan.

## 3) Evidence strength (1–5)
1: Mostly opinions and activities; few outcomes.  
3: Some outcomes and artifacts; evidence uneven.  
5: Strong evidence log with metrics/proxies, artifacts, and stakeholder proof per claim.

## 4) Narrative + skimmability (1–5)
1: Rambling doc; hard to scan.  
3: Reasonably structured; some strong bullets.  
5: Executive-friendly one-pager; thesis is clear; claims and evidence are easy to locate.

## 5) Next-level scope + leverage (1–5)
1: Work is scoped like current level; no leverage.  
3: Some next-level behaviors present; leverage mixed.  
5: Shows sustained next-level behaviors (ambiguity, cross-team influence, leverage, decision quality).

## 6) Manager alignment plan (1–5)
1: Manager not involved; no criteria or check-ins.  
3: Some alignment; criteria partially defined.  
5: Clear “Magic Loop” plan with agreed criteria, commitments, and scheduled check-ins.

## 7) Sponsor/visibility plan + risk handling (1–5)
1: No sponsor plan; ignores politics and risks.  
3: Basic stakeholder list; some outreach.  
5: Sponsor map with specific asks, no-surprises plan, and explicit risks/open questions/next steps.
