# meta-skills

## OVERVIEW
Skills about creating, reviewing, syncing, and managing skills themselves.

## SKILLS (6)
| Skill | Description |
|-------|-------------|
| `find-skills` | Discover and install agent skills |
| `lenny-skillpack-creator` | Convert Lenny skills to agent-executable skill pack format |
| `skill-creator` | Guide for creating new skills |
| `skill-review` | Skill quality review |
| `skills-readme-updater` | Update README.md from skill metadata |
| `sync-skills-manager` | Sync skills between repo and system directories |

## WHERE TO LOOK
| Task | Location |
|------|----------|
| Create a new skill | `skill-creator/SKILL.md` |
| Convert Lenny skill | `lenny-skillpack-creator/SKILL.md` |
| Sync to system dirs | `sync-skills-manager/SKILL.md` |

## CONVENTIONS
- Skill entrypoint is `SKILL.md`.
- Some skills include `scripts/` for automation (e.g., validation, README generation).
