# Pattern Library (workflow blueprints)

Pick 1–2 patterns per skill. Then tailor.

## Pattern A — Doc Generator

Outputs:
- A structured document (PRD, memo, strategy, plan, review)

Workflow:
1) Intake (goal, audience, constraints, success metrics)
2) Define problem + context (why/why now)
3) Define scope (goals, non-goals, assumptions)
4) Draft core sections (requirements / plan / arguments)
5) Add risks + mitigations + dependencies
6) Add rollout / timeline / owners
7) Quality gate + finalize

## Pattern B — Research & Synthesis

Outputs:
- Research plan + interview guide + synthesis + recommendations

Workflow:
1) Define research question + decision it informs
2) Choose methods (interviews, surveys, analytics, desk research)
3) Create instruments (interview guide, survey, data pulls)
4) Collect / summarize inputs
5) Synthesize into themes + evidence
6) Convert to recommendations + experiments
7) Quality gate

## Pattern C — Evaluation & Scoring

Outputs:
- Scorecard + rubric + recommendation + decision log

Workflow:
1) Define decision + options
2) Define criteria + weights
3) Score options with evidence
4) Sensitivity analysis (what changes the outcome)
5) Recommendation + rationale
6) Risks + mitigations
7) Quality gate

## Pattern D — Facilitation / Workshop

Outputs:
- Agenda + facilitation script + pre-read + notes template + follow-ups

Workflow:
1) Define objective + attendees + decisions needed
2) Pre-read (context, options, open questions)
3) Agenda with timing + prompts
4) Facilitation guide (what to do if stuck)
5) Capture template (decisions, owners, next steps)
6) Post-meeting follow-up message

## Pattern E — “AI Feature” Pack

Outputs (minimum):
- PRD/spec
- Eval spec (LLM-as-judge)
- Prompt set / prototype plan

Workflow:
1) Define user job + success metrics + constraints
2) Define non-negotiable behaviors (must / must-not)
3) Build test set (golden, adversarial, regression)
4) Define judge rubric + thresholds + gates
5) Prompt set / tool constraints / refusal policy
6) Monitoring + rollback
7) Quality gate
